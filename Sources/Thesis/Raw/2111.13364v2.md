# Prasad et al. (2021) — Optimal Technical Indicator-Based Trading Strategies Using NSGA-II

**Full citation:** Prasad, P.S.K., Madhav, V., Lal, R. & Ravi, V. (2021). Optimal Technical Indicator-based Trading Strategies Using NSGA-II. arXiv:2111.13364v2.

## Summary

Proposes using NSGA-II (Non-dominated Sorting Genetic Algorithm II), a bi-objective evolutionary algorithm, to find optimal combinations of technical indicators that simultaneously maximize Sharpe ratio and minimize maximum drawdown. Uses rolling-window training (2-year train, 1-year test). Results show considerably better performance than single-indicator strategies.

## Key Arguments

1. **Bi-objective optimization:** Maximize Sharpe ratio AND minimize max drawdown simultaneously — produces a Pareto front of optimal strategies.
2. **NSGA-II:** Population-based evolutionary search of the indicator-combination space; avoids local optima.
3. **Better than single indicators:** Combined indicator strategies outperform individual indicators, especially for stable periods.
4. **Rolling approach:** Avoids look-ahead bias through rolling window backtesting.
5. **Interpretability:** Evolutionary algorithms produce explicit rules (unlike neural nets), enabling insight into which indicator combinations work.

## Relevance to Thesis
