# Ke, Kelly & Xiu (2019) — Predicting Returns with Text Data [Working Paper]

**Full citation:** Ke, Z.T., Kelly, B. & Xiu, D. (2019). Predicting Returns with Text Data. BFI Working Paper 2019-69. Becker Friedman Institute, University of Chicago.

## Summary

Working paper version of the published Ke, Kelly & Xiu paper on supervised text-mining for return prediction. Content is essentially the same as the published SSRN version (ssrn-3389884). See [[ssrn-3389884]] for full annotation.

**Key findings (same as published):** Supervised sentiment scores adapted to return prediction outperform dictionary methods and commercial vendors (RavenPack) out-of-sample on Dow Jones Newswires data.
