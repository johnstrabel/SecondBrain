# Ohlson (1995) — Earnings, Book Values, and Dividends in Equity Valuation

**Full citation:** Ohlson, J.A. (1995). Earnings, Book Values, and Dividends in Equity Valuation. *Contemporary Accounting Research*, 11(2), 661–687.

## Summary

Foundational paper establishing the residual income (clean surplus) valuation model. Shows that firm value equals book value plus the present value of expected future abnormal earnings (earnings minus the cost of equity × book value). Requires only that clean surplus accounting holds (all value changes flow through earnings). This is the theoretical foundation for the Edwards-Bell-Ohlson (EBO) model.

*Note: PDF text extraction yielded minimal content (copyright restrictions); description based on well-known content of this seminal paper.*

## Key Arguments

1. **Clean surplus relation:** All changes in book value come from retained earnings; ensures dividend irrelevance.
2. **Abnormal earnings:** ROE − cost of equity applied to book value gives residual income; PV of future abnormal earnings equals the premium of market value over book value.
3. **Linear information dynamics (LID):** Abnormal earnings follow an AR(1) process; this allows closed-form valuation using only current book value, current earnings, and one parameter (persistence of abnormal earnings).
4. **Dividend irrelevance:** Given clean surplus, dividends are value-neutral (they reduce both book value and future earnings opportunities).
5. **Empirical operationalization:** Requires forecasts of earnings and book value — readily available from I/B/E/S.
