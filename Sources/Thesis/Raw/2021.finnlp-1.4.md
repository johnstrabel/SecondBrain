# Wang & Luo (2021) — Predicting $GME Stock Price Movement Using Sentiment from Reddit r/wallstreetbets

**Full citation:** Wang, C. & Luo, B. (2021). Predicting $GME Stock Price Movement Using Sentiment from Reddit r/wallstreetbets. In *Proceedings of the 3rd Workshop on Financial Technology and Natural Language Processing (FinNLP 2021)*.

## Summary

Workshop paper using WSB posts during the Jan 2021 GME short squeeze to predict daily price direction. Uses VADER sentiment + word2vec and BERT semantic embeddings; tests logistic regression, random forest, and deep learning classifiers. Unique challenge: WSB uses heavy emoji usage, memes, irony, slang that defeats standard NLP.

## Key Arguments

1. **Domain challenge:** WSB text has emojis, typos, irony, memes — standard NLP models struggle; custom preprocessing needed.
2. **VADER limitations:** Rule-based VADER misclassifies ironic posts (e.g., "retard" used positively in WSB community).
3. **BERT semantic features:** BERT last hidden layer (768-dim) as semantic representation; combined with VADER scores.
4. **Additional features:** Word count, stopword count, upvote score, comment count — metadata as sentiment proxies.
5. **Limited predictability:** Model achieves modest improvement over naive baseline — consistent with thesis ML AUC-ROC of 0.520.

## Relevance to Thesis

- Provides context for why Reddit NLP is difficult (domain mismatch — same issue as FinBERT on Reddit in thesis)
- GME-specific paper; thesis uses a 15-subreddit, 2.74M mention, multi-stock dataset — broader and more systematic
- Similar ML approach but at different scale and with different NLP tools

## Key Concepts

- [[Reddit as Financial Signal]] — WSB posts as prediction signal for GME
- [[FinBERT]] — BERT-family models for financial sentiment, though VADER used here not FinBERT
