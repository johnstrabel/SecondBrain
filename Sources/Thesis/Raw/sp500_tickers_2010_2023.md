# sp500_tickers_2010_2023.csv — Historical S&P 500 Constituent Tickers

## What This File Contains

636 historical S&P 500 constituent tickers covering 2010–2023. Includes tickers that entered and exited the index during this period — a survivorship-bias-free universe.

## Data Summary

- **N tickers:** 636 (vs. 500 at any point in time — includes additions and deletions)
- **Purpose:** Defines the universe for Reddit collection, price download, and EDGAR collection
- **Coverage:** All tickers that were S&P 500 members at any point 2010–2023

## Key Concepts

- [[Reddit as Financial Signal]] — Reddit collection targets this 636-ticker universe
- [[Post-Earnings Announcement Drift]] — event study restricted to 281 of these 636 with sufficient data
