# Zeng & Jiang (2023) — Financial Sentiment Analysis Using FinBERT with Application in Predicting Stock Movement

**Full citation:** Zeng, Q. & Jiang, T. (2023). Financial Sentiment Analysis Using FinBERT with Application in Predicting Stock Movement. arXiv:2306.02136v3.

## Summary

Integrates FinBERT for sentiment extraction from financial news with an LSTM network for sequence modeling to forecast stock market trends. Uses daily financial news + stock price data (merged via yfinance). Compares against standard BERT, standalone LSTM, and ARIMA. FinBERT+LSTM outperforms all alternatives. Sentiment score defined as P(positive) − P(negative).

## Key Arguments

1. **FinBERT for sentiment:** Uses ProsusAI/finbert — exact same model as in the thesis — to produce sentiment probabilities.
2. **Sentiment score formula:** SentScore = P(positive) − P(negative); aggregated daily per ticker.
3. **LSTM for sequence modeling:** Captures temporal dynamics in sentiment+price series.
4. **Outperforms baselines:** FinBERT+LSTM > BERT+LSTM > LSTM > ARIMA on stock movement prediction.
5. **News domain:** Financial news (formal text) — FinBERT performs better here than on informal social media.

## Relevance to Thesis

- Uses the identical FinBERT model (ProsusAI/finbert) as the thesis — confirms model selection
- Their SentScore = P(pos) − P(neg) is equivalent to thesis's net_sentiment feature
- News domain vs. Reddit domain — their better results may partly reflect fewer domain mismatch issues
