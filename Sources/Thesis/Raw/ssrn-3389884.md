# Ke, Kelly & Xiu (2021) — Predicting Returns with Text Data

**Full citation:** Ke, Z.T., Kelly, B. & Xiu, D. (2021). Predicting Returns with Text Data. *NBER Working Paper* / published version. (Working paper BFI_WP_201969 = 2019 draft)

## Summary

Introduces a text-mining methodology that extracts return-predictive sentiment from Dow Jones Newswires. Unlike dictionary-based methods, constructs a sentiment score **adapted to return prediction** via 3 steps: (1) predictive screening of sentiment terms, (2) topic-model weighting, (3) penalized likelihood aggregation. Outperforms RavenPack (commercial vendor) out-of-sample.

## Key Arguments

1. **Supervised vs. unsupervised sentiment:** Standard dictionaries (Harvard-IV, Loughran-McDonald) are not designed to predict returns — KKX learns weights from joint text-return behavior.
2. **Three-step method:** Predictive screening → topic modeling → penalized likelihood produces interpretable, parsimonious sentiment scores.
3. **Theoretical guarantees:** Error bounds derived under minimal assumptions; characterizes statistical limits of sentiment extraction.
4. **Fresh vs. stale news:** Article "novelty" measure shows fresh news drives more price impact than stale news.
5. **Out-of-sample:** Trading strategy (long positive sentiment, short negative) delivers alpha outperforming RavenPack.

## Relevance to Thesis

- Provides methodological precedent for using NLP-derived signals to predict returns (directly relevant to H1)
- Thesis uses pre-trained FinBERT (off-the-shelf) rather than a return-adapted model — KKX suggests this is suboptimal
- Explains why avg_sentiment_score has low ML feature importance (5.9%) relative to attention signals: FinBERT is not return-adapted
